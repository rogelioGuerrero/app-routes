Demo: Optimización Inteligente de Rutas

Escenario: 2 vehículos de reparto, cada uno con capacidad máxima de 3 paquetes. 4 clientes, cada uno con demanda de 1 paquete y preferencias horarias (ventanas de tiempo) para recibir su entrega. El sistema calcula la ruta óptima para cada vehículo, minimizando la distancia total y asegurando que todas las entregas se realicen en el horario preferido de cada cliente, respetando la capacidad de carga de cada vehículo.
Distancia total recorrida: 14.55 km

Datos de entrada

Vehículos: 2 unidades, cada una con capacidad máxima de 3 paquetes por recorrido.
Clientes: Cada cliente solicita 1 paquete y define su preferencia horaria para la entrega.

Nota sobre ventana de tiempo: El horario indica cuándo el cliente está disponible para recibir la entrega o visita. El sistema garantiza que cada entrega se realice dentro del rango solicitado, optimizando la puntualidad y satisfacción.
Nota sobre demanda: La demanda representa la cantidad a entregar en cada punto (pueden ser paquetes, cajas, pedidos, etc.). El sistema respeta la capacidad máxima de cada vehículo.

¿Por qué elegir nuestro optimizador?
  • Ahorro de tiempo y combustible: menos kilómetros recorridos.
  • Entregas puntuales: cumplimiento de ventanas de tiempo y entregas exactas.
  • Automatización total: solo ingresa los puntos y restricciones, el sistema hace el resto.
  • Escalable: soporta flotas y cientos de entregas.
  • Integración fácil: API lista para tu app, web o ERP.
